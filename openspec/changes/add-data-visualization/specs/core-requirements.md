# 核心需求规格

## ADDED 功能需求

### FR1: 分数趋势图
- **描述**: 展示历年录取分数变化趋势
- **验收标准**:
  - [ ] 支持折线图展示
  - [ ] 支持多条线对比（多所高校/专业）
  - [ ] 支持缩放和 hover 查看详情
  - [ ] 支持导出为 PNG

#### Scenario: 查看分数趋势
**Given**: 用户已选择某高校和专业
**When**: 用户查看结果详情
**Then**: 显示该专业近5年录取分数趋势图

### FR2: 高校对比雷达图
- **描述**: 多维度对比不同高校
- **验收标准**:
  - [ ] 支持雷达图展示
  - [ ] 至少5个维度（综合实力、学科排名、就业率、薪资、地理位置）
  - [ ] 支持选择2-5所高校对比
  - [ ] 支持导出

### FR3: 专业热度分布
- **描述**: 展示专业录取热度
- **验收标准**:
  - [ ] 支持柱状图或热力图
  - [ ] 按专业类别分组
  - [ ] 显示录取人数或分数分布

## MODIFIED 非功能需求

### NFR1: 性能
- 图表渲染时间 < 2秒
- 支持最多1000个数据点流畅展示

### NFR2: 兼容性
- 支持 Chrome、Safari、Firefox
- 支持移动端响应式
