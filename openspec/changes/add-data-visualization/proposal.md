# 提案：添加数据可视化功能

## Why

当前规划结果以文字和表格展示为主，缺乏直观的数据可视化，用户难以快速理解：
- 历年分数趋势
- 不同高校/专业的对比
- 自身成绩与目标线的差距

## What Changes

### 新增功能
- 分数趋势图（历年录取分数变化）
- 高校对比图（多维度雷达图）
- 专业热度图（词云/热力图）
- 学生成绩与目标差距图

### 新增文件
- `src/edu_info/ui/components/charts.py` - 图表组件库
- `src/edu_info/services/visualization_service.py` - 可视化服务

### 修改文件
- `src/edu_info/ui/pages/04_results_display.py` - 集成图表展示

## 背景

项目已有完整的数据层和规划引擎，但 UI 展示以表格和文字为主。添加可视化可以提升用户体验。

## 目标

1. 使用 Plotly 实现交互式图表
2. 图表支持筛选和缩放
3. 支持导出图表为图片

## 成功标准

1. 结果展示页面显示至少 3 种图表
2. 图表可交互（hover、缩放）
3. 支持导出图表为 PNG
