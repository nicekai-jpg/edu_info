# 架构规格

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        用户界面层                            │
│                    (Streamlit Web UI)                       │
├─────────────────────────────────────────────────────────────┤
│                        业务逻辑层                            │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ 学生管理    │  │ 数据分析    │  │ 规划推荐引擎        │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
├─────────────────────────────────────────────────────────────┤
│                        数据访问层                            │
│                    (DuckDB Database)                        │
├─────────────────────────────────────────────────────────────┤
│                        数据层                                │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ 高校数据    │  │ 专业数据    │  │ 录取分数数据        │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | Streamlit | latest |
| 后端 | Python | 3.12+ |
| 数据库 | DuckDB | latest |
| 包管理 | uv | latest |
| 数据验证 | Pydantic | v2 |
| 数据处理 | Pandas | latest |

## 模块职责

### src/edu_info/
- `main.py` - Streamlit 应用入口
- `models/` - Pydantic 数据模型
- `services/` - 业务逻辑服务
- `database/` - 数据库访问层
- `utils/` - 工具函数

### 数据流
1. 用户通过 Streamlit UI 输入数据
2. 数据经过 Pydantic 模型验证
3. 业务逻辑层处理数据
4. DuckDB 存储/查询数据
5. 结果返回给用户
