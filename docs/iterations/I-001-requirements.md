## 1. 上下文

### 业务背景
升学规划咨询系统的第一步是**建立完整的数据基础**。本迭代I-001负责数据层建设，包括：
1. 全国高校基础数据（985/211/双一流等）
2. 专业目录数据（学科门类、专业类、具体专业）
3. 辽宁录取分数数据（2021-2025年，含位次）
4. 政策相关数据（招生政策、特殊类型招生）

### 技术背景
- 数据库：DuckDB（嵌入式、高性能分析）
- 数据导入：支持Excel/CSV（历史数据迁移）
- 数据采集：预留阳光高考等数据源接口
- 数据管理：基础查询和预览功能

## 4. 范围

### 包含（In Scope）

1. **项目目录结构**（src/data/, src/utils/, data/, logs/, config/）

2. **数据库设计**（6张核心表）：
   - `universities` - 高校基础信息
   - `majors` - 专业目录信息
   - `admission_scores` - 录取分数数据（核心）
   - `admission_policies` - 招生政策数据
   - `special_admissions` - 特殊类型招生（强基/综评/特长生等）
   - `data_sources` - 数据来源记录（追溯性）

3. **数据导入功能**：
   - 高校基础数据导入（代码、名称、层次、省份等）
   - 专业数据导入（学科门类、专业类、专业代码等）
   - 录取分数数据导入（年份、科类、分数、位次等）
   - 支持格式：.xlsx（优先）、.csv
   - 数据验证：必填字段检查、类型校验、外键引用检查

4. **数据采集接口**（预留）：
   - 阳光高考网数据抓取接口（预留，受反爬限制可能手动）
   - 辽宁招生考试之窗接口（预留）
   - 各高校招生网接口（预留）
   - 统一的数据采集器抽象基类

5. **基础数据管理**：
   - 高校列表查询（支持按层次、省份、类型筛选）
   - 专业列表查询（支持按学科门类筛选）
   - 录取分数查询（支持按年份、科类、分数段筛选）
   - 数据预览（表格展示，分页）
   - 数据统计（高校数量、专业数量、数据覆盖年份等）

6. **数据质量保障**：
   - 导入错误提示（行号、字段、错误类型）
   - 数据重复检测（同一高校同一专业同一年份）
   - 数据完整性检查（必填字段非空）
   - 数据来源记录（可追溯）

## 5. 假设

1. 数据来源主要是历史Excel/CSV文件（从阳光高考等平台手动下载）
2. 数据格式相对标准（字段名可能不同，但内容格式规范）
3. 辽宁录取数据为优先（物理类/历史类，2021-2025）
4. 数据量：高校<2000所，专业<1000个，分数线<100万条
5. 单用户使用，无并发写入需求

## 6. 约束

### 性能约束
- 数据导入：1万条 < 10秒
- 简单查询：< 1秒
- 复杂筛选（多条件）：< 3秒
- 数据库文件大小：< 500MB（含索引）

### 技术约束
- Python 3.11+
- DuckDB 1.1+
- pandas 2.0+
- 使用 SQLAlchemy 或 DuckDB Python API

### 数据约束
- 录取分数必须包含**位次**（关键指标，比分数更稳定）
- 支持物理类/历史类（新高考模式）
- 支持文科/理科（旧高考模式，2021年数据）
- 年份范围：2021-2025（近5年）

### 平台约束
- 本地运行，支持 Windows/macOS/Linux
- 数据存储在本地文件系统

## 10. 边界情况/故障模式

### 边界情况

| 场景 | 输入 | 预期行为 |
|------|------|---------|
| 边界1 | 空文件（只有表头） | 导入成功，数量0，不报错 |
| 边界2 | 超大文件（10万+行） | 分批导入，每批1000条，显示进度 |
| 边界3 | 特殊字符（emoji、生僻字） | 正常存储，UTF-8编码 |
| 边界4 | 缺失位次数据 | 标记警告，允许导入但提示"位次为空" |
| 边界5 | 新旧高考科类混合（2021文科/理科，2022+物理/历史） | 统一映射到 subject_type 字段 |

### 故障模式

| 故障 | 触发条件 | 处理方式 |
|------|---------|---------|
| 故障1 | 文件格式错误（非Excel/CSV） | 拒绝导入，提示支持的格式 |
| 故障2 | 编码错误（非UTF-8/GBK） | 尝试chardet检测，失败时提示 |
| 故障3 | 必填字段缺失 | 显示错误行号和缺失字段 |
| 故障4 | 外键引用不存在（高校/专业不存在） | 显示错误行和引用ID，跳过或拒绝 |
| 故障5 | 重复数据（同校同专业同年同科类） | 提示重复，提供"跳过/覆盖/保留两者"选项 |
| 故障6 | 数据库文件被占用 | 提示"数据库被占用"，建议关闭其他程序 |
