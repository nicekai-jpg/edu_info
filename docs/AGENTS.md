## 项目概述

基于**国家战略发展方向**，整合**教育政策**与**高校录取数据**，为初中/高中学生提供个性化升学规划方案的本地运行工具。

### 核心数据流
```
国家战略 → 政策分析 → 路线穷举 → 学生匹配 → 目标生成 → 可行性评估
```

---

## 业务模块

| 模块 | 职责 | 数据来源 |
|------|------|----------|
| **数据采集** | 收集全国高校在辽宁录取数据、政策文件 | 阳光高考、教育官网 |
| **路线穷举器** | 穷举所有可能的升学路线 | 政策文件、招生简章 |
| **匹配引擎** | 多维度匹配学生与路线 | 算法模型 |
| **目标生成器** | 为每条路线生成高/中/低三档目标 | 分数数据 |
| **本地UI** | 学生档案管理、规划分析、结果展示 | Streamlit |

## 规划逻辑

```
1. 穷举路线：生成50-100条可能的升学路线
   - 普通高考（985/211/普通本科）
   - 科技特长生（信息学/机器人/科创）
   - 艺术特长生（音乐/美术/舞蹈）
   - 体育特长生
   - 拔尖人才（奥赛/强基/综评）
   - 特殊类型（港澳/中外合作）

2. 多维度匹配：
   - 兴趣匹配（40%）
   - 能力匹配（25%）
   - 经济匹配（20%）
   - 时间匹配（10%）
   - 地域匹配（5%）

3. 生成三档目标：
   - 高目标（冲刺）
   - 中目标（稳妥）
   - 低目标（保底）

4. 可行性评估 + 备选方案

5. 方案存储与更新：
   - 存储完整规划方案
   - 记录生成时间和数据版本
   - 支持定期重新生成
   - 版本对比与变更追踪
```
