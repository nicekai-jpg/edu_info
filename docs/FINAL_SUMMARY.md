## 🚀 使用方法

### 1. 安装依赖

```bash
cd /Users/limingkai/nas/project/edu_info
uv sync
```

### 2. 收集数据

```bash
python3 scripts/collect_data.py
```

### 3. 初始化数据库

```bash
python3 scripts/setup_db.py
```

### 4. 启动应用

```bash
uv run streamlit run src/edu_info/main.py --server.port=8501
```

访问 http://localhost:8501

### 5. 使用流程

1. **创建学生档案** - 在"学生档案"页面
2. **运行规划分析** - 在"规划分析"页面
3. **查看结果** - 查看三档目标高校和推荐路线
