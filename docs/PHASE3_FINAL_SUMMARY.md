### 3. 在代码中使用

```python
from edu_info.core.planning_engine import PlanningEngine
from edu_info.models.schemas import Student

# 创建学生
student = Student(...)

# 创建引擎
engine = PlanningEngine()

# 生成规划
result = engine.generate_plan(student)

# 查看结果
print(f"推荐路线：{result.top_routes}")
print(f"高目标：{result.high_targets}")
print(f"中目标：{result.medium_targets}")
print(f"低目标：{result.low_targets}")
```
