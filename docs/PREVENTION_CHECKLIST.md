### 测试编写规范

- [ ] **Fixture 管理**
  ```python
  # ✅ 正确：只生成路径，不创建文件
  @pytest.fixture
  def temp_db():
      db_path = tempfile.mktemp(suffix='.db')
      yield db_path
      if os.path.exists(db_path):
          os.remove(db_path)
  
  # ❌ 错误：预先创建文件导致问题
  @pytest.fixture
  def temp_db():
      with tempfile.NamedTemporaryFile() as f:
          yield f.name  # 文件已存在
  ```

- [ ] **测试命名**
  - [ ] `test_功能_场景_预期结果` 格式
  - [ ] 清晰表达测试意图
  - [ ] 避免过长的测试名

- [ ] **断言质量**

1. **CI/CD 集成**自动检查项目
