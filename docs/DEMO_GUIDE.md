## 📋 页面功能演示

### 1️⃣ 🏠 首页

**功能**:
- 系统介绍
- 功能导航
- 快速入口

**操作**:
1. 打开首页查看系统介绍
2. 点击导航按钮跳转到对应功能页面

---

### 2️⃣ 📝 学生档案管理

**功能**:
- 查看现有学生档案
- 创建新学生档案
- 编辑学生信息
- 删除学生档案

**操作步骤**:

#### 查看学生档案
1. 点击左侧菜单 "📝 学生档案"
2. 查看现有学生列表（已有 2 个示例学生）
3. 点击学生姓名查看详情

#### 创建新学生
1. 点击 "➕ 新建学生档案" 按钮
2. 填写学生信息：
   - 基本信息：姓名、性别、出生日期
   - 学籍信息：当前年级、选科类型
   - 成绩信息：各科成绩、总分
   - 兴趣偏好：兴趣领域、 preferred 地区等
3. 点击 "💾 保存" 按钮

#### 编辑学生
1. 在学生列表中选择要编辑的学生
2. 点击 "✏️ 编辑" 按钮
3. 修改信息后保存

---

### 3️⃣ 📥 数据导入

**功能**:
- 导入高校数据
- 导入学生档案
- 导入录取分数

**操作步骤**:

#### 导入高校数据
1. 点击 "📥 数据导入" 页面
2. 选择 "🏫 高校数据" 标签页
3. 上传 Excel 或 JSON 文件
4. 预览数据
5. 点击 "导入到数据库"

#### 导入学生档案
1. 选择 "👨‍🎓 学生档案" 标签页
2. 上传学生数据文件
3. 预览并确认
4. 导入到数据库

#### 导入录取分数
1. 选择 "📊 分数线数据" 标签页
2. 上传分数数据文件
3. 选择年份（2021-2025）
4. 导入到数据库

---

### 4️⃣ 📊 规划分析

**功能**:
- 选择学生
- 运行规划引擎
- 生成规划方案

**操作步骤**:

1. **选择学生**
   - 从下拉列表中选择要分析的学生
   - 查看学生基本信息

2. **配置分析参数**（可选）
   - 选择规划类型
   - 设置匹配权重
   - 选择目标高校范围

3. **运行分析**
   - 点击 "🚀 开始规划分析" 按钮
   - 等待引擎计算（通常 1-3 秒）

4. **查看分析结果**
   - 系统自动跳转到结果页面
   - 查看推荐的规划路线

---

### 5️⃣ 📤 结果展示

**功能**:
- 规划总结
- 推荐路线 Top 10
- 三档目标高校
- 可行性评估
- 行动建议
- 导出功能

**结果详情**:

#### 规划总结
- 分析的学生信息
- 生成的路线数量
- 匹配度分布

#### 推荐路线 Top 10
每条路线包含：
- 路线名称和类型
- 匹配度评分
- 适合理由
- 目标高校（高/中/低三档）

#### 三档目标高校
- **高目标**（冲刺）：匹配度 80-90%
- **中目标**（稳妥）：匹配度 60-80%
- **低目标**（保底）：匹配度 40-60%

#### 可行性评估
- 学术可行性
- 经济可行性
- 时间可行性
- 综合评分

#### 行动建议
- 短期目标（1 年内）
- 中期目标（2-3 年）
- 长期目标（3-5 年）

#### 导出功能
1. 点击 "📥 导出结果"
2. 选择格式：
   - JSON（完整数据）
   - CSV（表格数据）
3. 保存到本地

## 🎬 完整操作流程示例

### 场景：为高二学生张三做升学规划

#### 步骤 1: 查看/创建学生档案
```
1. 进入 "📝 学生档案" 页面
2. 查看现有学生列表
3. 如果张三已存在，点击查看详情
4. 如果不存在，点击 "➕ 新建" 创建档案
```

#### 步骤 2: 运行规划分析
```
1. 进入 "📊 规划分析" 页面
2. 从下拉列表选择 "张三"
3. 点击 "🚀 开始规划分析"
4. 等待 1-3 秒
```

#### 步骤 3: 查看规划结果
```
1. 系统自动跳转到 "📤 结果展示" 页面
2. 查看规划总结
3. 浏览推荐路线 Top 10
4. 查看三档目标高校
5. 阅读可行性评估
6. 记下行动建议
```

#### 步骤 4: 导出结果
```
1. 点击 "📥 导出结果"
2. 选择 JSON 或 CSV 格式
3. 保存到本地
4. 可以打印或分享给家长
```

## 🔧 常见问题

### Q1: 如何添加新学生？
**A**: 进入 "📝 学生档案" → 点击 "➕ 新建" → 填写信息 → 保存

### Q2: 规划分析需要多长时间？
**A**: 通常 1-3 秒，取决于数据量和复杂度

### Q3: 如何更新录取分数数据？
**A**: 进入 "📥 数据导入" → 选择 "📊 分数线数据" → 上传新数据

### Q4: 导出的文件在哪里？
**A**: 默认保存到浏览器的下载文件夹

### Q5: 如何清空数据重新开始？
**A**: 
```bash
# 备份数据库（可选）
cp data/duckdb/edu_planning.db backup.db

# 删除数据库
rm data/duckdb/edu_planning.db

# 重新初始化
uv run python scripts/init_database.py
```
