## 核心设计理念

### 从"分数匹配"到"趋势预判"

```
传统思路（被动）：
高考分数 → 查往年数据 → 匹配院校 → 填报志愿

本系统思路（主动）：
国家趋势 → 识别突破口 → 匹配政策 → 选择专业 → 确定高校 → 制定路径 → 持续调整
```

### 三档目标体系

```
高目标（理想）：
- 最好的专业方向
- 顶尖高校
- 需要极大努力
- 失败风险较高

中目标（务实）：
- 较好的专业方向
- 中上高校
- 需要持续努力
- 相对稳妥

低目标（保底）：
- 可接受的专业方向
- 一般高校
- 基本可达成
- 确保有学上
```

### 全链条覆盖

```
初中 → 高中 → 高考 → 大学 → 就业
  │      │      │      │      │
  ▼      ▼      ▼      ▼      ▼
兴趣培养 特长发展 多元升学 专业深造 职业规划
```

## 数据收集范围（核心）

### 1. 宏观趋势数据

- **国家战略**：十四五/十五五规划、中国制造2025、战略性新兴产业目录
- **产业趋势**：未来5-10年重点发展领域、人才缺口预测
- **区域规划**：全国区域发展布局（京津冀、长三角、珠三角、成渝等）、区域产业布局、人口流动趋势

### 2. 教育政策数据

- **课改政策**：新课程方案、学科素养要求、教材改革
- **评价改革**：综合素质评价、学业水平考试
- **升学政策**：
  - 普通高考（统招）
  - 艺术特长生（音乐、美术、舞蹈）
  - 科技特长生（信息学、机器人、科创）
  - 体育特长生
  - 保送生（奥赛、外语类）
  - 强基计划
  - 综合评价
  - 高校专项计划
  - 高水平艺术团/运动队

### 3. 全国高校在辽宁录取数据

- **985高校**：39所在辽宁录取分数
- **211高校**：116所在辽宁录取分数
- **双一流高校**：147所在辽宁录取分数
- **其他重点高校**：行业特色高校、省属重点等
- **专业信息**：专业目录、学科评估、就业前景
- **分数线**：近5年各专业录取分数、位次（关键指标）
- **特殊招生**：各校特长生、综评等招生简章

## 规划分析流程

```
输入：学生基本情况（年级、成绩、兴趣、家庭条件）
  │
  ▼
┌─────────────────────────────────────┐
│ Step 1: 突破口识别                   │
│ 分析国家战略 + 产业趋势 + 人才缺口    │
│ 输出：3-5个未来潜力方向              │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ Step 2: 政策匹配                     │
│ 为每个突破口找到支持政策              │
│ 识别适用的升学途径                    │
│ 输出：政策-途径映射表                │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ Step 3: 专业选择                     │
│ 匹配突破口相关的专业                  │
│ 评估专业前景与个人兴趣                │
│ 输出：推荐专业列表                   │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ Step 4: 高校确定                     │
│ 筛选专业实力强的高校                  │
│ 分析历年录取数据                      │
│ 输出：目标高校（高/中/低三档）       │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ Step 5: 可行性评估                   │
│ 对比当前水平与目标要求                │
│ 识别能力差距                          │
│ 评估达成概率                          │
│ 输出：可行性报告                     │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ Step 6: 备选方案                     │
│ 高目标不可行时 → 降低高校层次        │
│ 中目标不可行时 → 调整专业方向        │
│ 输出：备选方案（同专业不同高校）     │
└──────────────┬──────────────────────┘
               │
               ▼
输出：完整规划报告（含三档目标+行动建议）
```
